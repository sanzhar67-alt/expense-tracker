# 💰 Қазына - Жеке қаржы трекері

Каржы трекері - кіріс/шығыс бақылау, валюта конвертациясы, диаграммалар.

## 🌐 Тірі нұсқа
**[Жобаны көру](https://sanzhar67-alt.github.io/expense-tracker/)**

## 📱 Функционалдар

- ✅ Кіріс / шығыс қосу және жою
- ✅ Автоматты баланс есептеу
- ✅ Қараңғы / Жарық тақырып
- ✅ Валюта конвертациясы (KZT, USD, EUR, RUB)
- ✅ Шығыс диаграммасы (Pie Chart)
- ✅ LocalStorage (деректер сақталады)
- ✅ Толық адаптив (мобильді құрылғыларға бейімделген)

## 🛠 Технологиялар

| Технология | Қолданылуы |
|------------|-------------|
| HTML5 | Құрылым |
| CSS3 | Стильдеу (Flexbox/Grid, BEM) |
| JavaScript | Логика, State management |
| Chart.js | Круговая диаграмма |
| ExchangeRate API | Валюта курсстары |
| LocalStorage | Деректерді сақтау |
| GitHub Pages | Хостинг |
